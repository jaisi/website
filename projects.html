My Sugar Log

My Sugar Log is a scalable, single-page JavaScript application to keep track of a user's blood sugar values. It is built in the Angular framework. It is a simple CRUD application, and uses visualization tools like graphs to give the user a quick picture of their values over several periods of time. It calculates the A1c of the user based on values entered. The user also has the ability to save the readings and graphs as a pdf or take printouts. The purpose of this application is to provide an easy interface to diabetics to enter their blood sugars and keep track of their health in a non invasive and least complicated manner.
